Object Oriented Style¶

Many aspects of the Object Oriented Style figures are the same as the STATEFUL figures. But the way we access or apply functions to them are different!

  • step 1: Define the figure and axes objects - create variables! 定义图形和轴对象
  • step 2: plot the data within the axes using methods associated with the axes objects! 使用与坐标区对象关联的方法在坐标区内绘制数据！
  • step 3: modify teh axes object attributes if needed 修改轴对象
  • step 4: modify the figure object attributes if needed 修改图形对象
  • step 5: show the figure 显示图

The motivation for the object oriented approach is that the STATEFUL approach works with HIDDEN OBJECTS!!!!

Define variables or objects for the FIGURE and AXES that we are modifying!

The common names for these objects are fig and ax, the type of ax is numpy array, and modifying it doesn't show the plot. The type of fig is matplotlib.figure.Figure, which shows the figure.
